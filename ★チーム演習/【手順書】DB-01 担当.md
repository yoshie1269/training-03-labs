# DB-01 担当 詳細設計・構築手順書

担当者：[ ]
サーバ名：db01.wp.local

---

## 1. 役割
- MariaDB
- Dovecot
- NTP
- NFS

---

## 2. MariaDB構築

dnf install -y mariadb-server

systemctl enable --now mariadb

DB作成：
CREATE DATABASE banana_db;

---

## 3. Dovecot構築

dnf install -y dovecot

mail_location = maildir:~/Maildir

---

## 4. NTP構築

dnf install -y chrony

server ntp.wp.local iburst

---

## 5. NFS構築

dnf install -y nfs-utils

/etc/exports 設定：
/var/www *(rw,sync)

---

## 6. 共有設定

exportfs -ra

---

## 7. 確認

| 項目 | コマンド |
|------|----------|
| DB | mysql -u root |
| Mail | ls ~/Maildir |
| NTP | chronyc sources |
| NFS | showmount -e |

---

## 8. 障害対応

- DB停止：
  systemctl restart mariadb
- NFS不可：
  exportfs -v
- Mail不可：
  systemctl restart dovecot
