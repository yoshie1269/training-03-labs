# nginx インストール・起動確認 手順書（Amazon Linux 2023）

---

# 概要

本手順では Amazon Linux 2023 環境に nginx をインストールし、  
Webサーバとして起動・動作確認まで行う。

実施内容
```bash
1 nginx インストール  
2 nginx バージョン確認  
3 nginx ビルドオプション確認  
4 nginx サービス起動  
5 ポート待ち受け確認  
```
---

# 構成イメージ
```bash
ユーザ
↓
HTTPリクエスト
↓
EC2 (Amazon Linux 2023)
↓
nginx
↓
/var/www/html/index.html
```
---

# 1 rootユーザへ切り替え

## 操作サーバ
EC2 (Webサーバ)

## この工程でしていること
管理者権限で作業するため root に切り替える

## 実行コマンド
```bash
sudo su -
```
## コマンドの意味

| コマンド | 意味 |
|---|---|
| sudo | 管理者権限で実行 |
| su | ユーザ切替 |
| - | rootログインシェル |

---

# 2 nginx インストール

## この工程でしていること
Webサーバ nginx をOSへインストールする

## 実行コマンド
```bash
dnf install nginx -y
```
## コマンドの意味

| コマンド | 意味 |
|---|---|
| dnf | パッケージ管理 |
| install | ソフトインストール |
| nginx | Webサーバ |
| -y | 確認省略 |

---

## インストールされたパッケージ

| パッケージ | 役割 |
|---|---|
| nginx | nginx本体 |
| nginx-core | コア機能 |
| nginx-filesystem | ディレクトリ構造 |
| nginx-mimetypes | MIMEタイプ |
| gperftools-libs | パフォーマンス最適化 |
| libunwind | スタック解析 |
| generic-logos-httpd | Apache互換ロゴ |

---

## OKの目安
```bash
Complete!
```
表示されれば成功

---

| コマンド | 意味 |

| nginx | nginx実行 |
| -v | バージョン表示 |


# 4 nginx ビルド情報確認

## この工程でしていること
nginx のコンパイルオプション確認

## 実行コマンド
```bash
nginx -V
```
## コマンドの意味

| オプション | 意味 |
|---|---|
| -V | 詳細情報表示 |

---

## 確認できる内容

| 項目 | 意味 |
|---|---|
| OpenSSL | SSLライブラリ |
| TLS SNI | 複数証明書 |
| modules-path | モジュールパス |
| conf-path | 設定ファイル |
| error-log-path | エラーログ |
| access-log-path | アクセスログ |

---

## nginx主要ディレクトリ

| パス | 用途 |
|---|---|
| /etc/nginx/nginx.conf | 設定 |
| /usr/sbin/nginx | 実行バイナリ |
| /var/log/nginx | ログ |
| /usr/share/nginx/html | Webコンテンツ |

---

# 5 nginx サービス状態確認

## この工程でしていること
nginx が起動しているか確認

## 実行コマンド
```bash
systemctl status nginx.service
```
## コマンドの意味

| コマンド | 意味 |
|---|---|
| systemctl | systemd管理 |
| status | 状態確認 |
| nginx.service | nginxサービス |

---

## 確認結果

Active: inactive (dead)

意味

nginx はまだ起動していない

---

# 6 nginx 起動

## この工程でしていること
nginx サービスを起動する

## 実行コマンド
```bash
systemctl start nginx.service
```
## コマンドの意味

| コマンド | 意味 |
|---|---|
| start | サービス開始 |
| nginx.service | nginx |

---

# 7 nginx 起動確認

## 実行コマンド
```bash
systemctl status nginx.service
```
## OKの目安

Active: active (running)

表示される

---

# 8 自動起動設定（正しいコマンド）

## 実行コマンド
```bash
systemctl enable nginx.service
```
---

## コマンドの意味

| コマンド | 意味 |
|---|---|
| enable | OS起動時自動起動 |

---

# 9 ポート待ち受け確認

## この工程でしていること
nginx が80番で待ち受けているか確認

## 実行コマンド
```bash
ss -tulpn | grep nginx
```
---

## コマンドの意味

| コマンド | 意味 |
|---|---|
| ss | ソケット確認 |
| -t | TCP |
| -u | UDP |
| -l | LISTEN |
| -p | プロセス |
| -n | 数値表示 |

---

## 実行結果
```bash
0.0.0.0:80
[::]:80
```
意味

| 表示 | 意味 |
|---|---|
| 0.0.0.0:80 | IPv4 |
| [::]:80 | IPv6 |

---

## nginxプロセス
```bash
nginx: master process
nginx: worker process
```
意味

| プロセス | 役割 |
|---|---|
| master | 管理 |
| worker | 実処理 |
